# <img src="./icons/elmtrn.png" width="32" height="32" /> elmtrn

elm + electron boiler plate

# Requirement

- elm (v 0.19)
- npm

# Quickstart

```
git clone https://github.com/yasuyuky/elmtrn
cd elmtrn
npm install -g electron
npm install
npm start
```
